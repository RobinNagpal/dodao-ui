# Downside Analysis

Equity downside / drawdown framework with per-ticker case studies — the consolidated 31-stock study used to derive the final downside-evaluation framework.

## Subfolders

- **[price_dip_analysis/](price_dip_analysis/)** — The 31-stock study itself: per-ticker analysis files (one `<TICKER>.md` per stock, e.g. `INVZ.md`, `PRKS.md`, `OBDC.md`, …) plus four cross-cutting reference files described below.

### Reference files inside `price_dip_analysis/`

- **[price_dip_analysis/frameworks_and_methodology.md](price_dip_analysis/frameworks_and_methodology.md)** — The 7 downside-analysis frameworks, side-by-side comparison, known shortcomings, and the methodology shared across all 31 case studies.
- **[price_dip_analysis/comparative_analysis.md](price_dip_analysis/comparative_analysis.md)** — Cross-stock comparative analysis pulling signals out of the 31-stock corpus.
- **[price_dip_analysis/ratio_analysis.md](price_dip_analysis/ratio_analysis.md)** — Financial-ratio reference: ranges, unit definitions, and expected price sensitivity for each ratio used in the framework.
- **[price_dip_analysis/final_framework_for_downside.md](price_dip_analysis/final_framework_for_downside.md)** — The synthesized "How to evaluate downside risk in stocks" framework derived from the study — the reader-facing version.
