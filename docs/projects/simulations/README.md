# Simulations

An educational web application for interactive business case study experiences. Students work through real-world business scenarios across multiple disciplines; instructors create cases, assign them, and grade submissions; AI provides contextual hints and feedback.

## Subject Areas

The platform covers five core business disciplines:

- **Marketing** — market analysis & strategy, consumer behavior, brand management, digital campaigns
- **Finance** — financial analysis & planning, investment decisions, risk assessment, corporate finance
- **Human Resources** — talent management, organizational behavior, performance management, conflict resolution
- **Operations** — supply chain, process optimization, quality control, logistics & distribution
- **Economics** — market dynamics, policy analysis, competitive strategy, industry analysis

## User Roles

### Students
- Access assigned case studies
- Work through business scenarios
- Submit solutions and analyses
- Receive AI-powered feedback
- Track progress and performance

### Instructors
- Create and manage case studies
- Assign studies to students
- Monitor student progress
- Provide feedback and grading
- Generate performance reports

### Administrators
- Manage users and permissions
- Oversee platform operations
- Configure system settings
- Access analytics and reporting

## Workflow

### Student journey
1. **Login** — secure authentication
2. **Dashboard** — view assigned case studies
3. **Case Study** — interactive scenario engagement
4. **Exercise** — practical application tasks
5. **Submission** — solution submission
6. **Summary** — performance review and feedback

### Instructor journey
1. **Case Creation** — design new scenarios
2. **Student Assignment** — distribute to learners
3. **Progress Monitoring** — track engagement
4. **Assessment** — review and grade submissions
5. **Feedback** — provide detailed guidance

## AI Integration

- **Intelligent tutoring** — contextual hints, personalized learning paths, automated feedback
- **Content generation** — dynamic scenario variations, question generation, assessment rubrics

## Tech Stack

- **Framework**: Next.js 15 with React 18 (App Router)
- **Language**: TypeScript
- **UI**: Tailwind CSS, Radix UI primitives, `@uiw/react-md-editor` for rich text
- **Database**: PostgreSQL with Prisma ORM
- **Authentication**: NextAuth.js with Prisma adapter
- **AI**: Google GenAI for tutoring + content generation
