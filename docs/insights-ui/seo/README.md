# SEO — Indexing Audits

Per-cohort audits of KoalaGains URLs that fail Google Search Console validation
("Crawled — currently not indexed", "Discovered — currently not indexed", duplicate-
canonical variants, Soft 404). Each file analyses one cohort, identifies shared
patterns, and proposes targeted fixes.

## Files

- **[failed-indexed-future-performance-2026-06-17.md](failed-indexed-future-performance-2026-06-17.md)** —
  Audit of the 27 `future-performance` URLs that failed GSC re-validation on
  2026-06-17. Identifies six root-cause patterns (recent-bulk-batch hold, non-
  public / mismatched (ticker, company) pairs, anomalous factor counts, all-fail
  factor blocks, narrative-heavy industries, missing non-US breadcrumb) with
  per-URL findings and a fix-priority order.
